# 🏥 Hospital Management System

A **Hospital Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. This project simplifies hospital operations by providing a secure and user-friendly platform for patients and administrators. Patients can register, log in, book doctor appointments, and manage their profiles, while administrators can efficiently manage doctors, appointments, and hospital records through a dedicated dashboard.

## ✨ Features

- 👤 Patient Registration & Login
- 🔐 Secure JWT Authentication
- 📅 Online Doctor Appointment Booking
- 👨‍⚕️ Doctor & Department Management
- 📋 Appointment Management
- 🛠️ Admin Dashboard
- 📱 Fully Responsive User Interface
- ⚡ RESTful API with Express.js
- 💾 MongoDB Database Integration

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone <repository-url>
cd Hospital-Management-System
```

---

## 2. Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

### Dashboard

```bash
cd dashboard
npm install
```

---

## 3. Run the Project

### Start Backend Server

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```

### Start Admin Dashboard

```bash
cd dashboard
npm run dev
```

---

## 4. Open in Browser

- **Frontend: ** hospitalfrontendmanagement.netlify.app
- **Admin Dashboard: ** hospitaladmindashboard.netlify.app
- **Backend API:** https://hospital-management-system-psg0.onrender.com
